# Workspace Directory
After the initial setup your `~/workspace` directory should look like this:
```bash
|-- workspace
    |-- ros2_ws
        |-- build
        |-- install
        |-- log
        |-- src
            |-- calibration
            |-- imu_calib
            |-- ldrobot-lidar-ros2
            |-- peripherals
            |-- driver
            |-- orchestrator_launch
            |-- simulations
    |-- software
        |-- Servo_upper_computer
        |-- collect_picture
        |-- lab_tool
        |-- labelImg
        |-- tool
        |-- yolov5
```
This chapter provides an overwiew over the different folders and what they implement respectively.
